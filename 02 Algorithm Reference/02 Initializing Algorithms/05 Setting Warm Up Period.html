Often algorithms need some historical data to prime technical indicators, or populate historical data arrays. Using the $[SetWarmUp(),M:QuantConnect.Algorithm.QCAlgorithm.SetWarmUp] method you can specify a warm up period for your algorithm which pumps data in from before the start date. During the warm up period you cannot place a trade.

Algorithms can use the $[bool IsWarmingUp,P:QuantConnect.Algorithm.QCAlgorithm.IsWarmingUp] property to determine if the warm up period has completed.

See more about using historical data in the History section. 

SetWarmUp(200); //Warm up 200 bars for all subscribed data.
SetWarmUp(TimeSpan.FromDays(7)); //Warm up 7 days of data.

self.SetWarmUp(200) # Warm up 200 bars for all subscribed data.
self.SetWarmUp(timedelta(7)) # Warm up 7 days of data.

this.SetWarmUp(200)
